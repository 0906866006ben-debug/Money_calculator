# 倉位風控計算機

純本機離線 PWA。輸入總資金，用 +/- 按鈕調整止損距離，即時計算建議槓桿與每單保證金。

規則固定：

- 每單保證金 = 總資金 2%
- 目標 1R = 總資金 1%
- 止損距離每次調整 0.1%，顯示到小數點後一位
- 槓桿依止損距離反推，並向下取整

## 本機開發

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

開啟：

```text
http://127.0.0.1:4173/
```

## 手機安裝

手機瀏覽器安裝 PWA 與 Service Worker 需要安全來源。正式使用時，把整個資料夾部署到 HTTPS 靜態站台，例如 GitHub Pages、Cloudflare Pages、Netlify 或自己的 HTTPS 主機。

部署後用手機瀏覽器開啟網址，選「加入主畫面」。第一次載入完成後，Service Worker 會快取靜態檔，之後可離線開啟與計算。

## 驗收數值

- 總資金 100U、止損 4.8% -> 保證金 2.00U，建議 10x，理論 10.42x。
- 總資金 100U、止損 5.5% -> 保證金 2.00U，建議 9x，理論 9.09x。
